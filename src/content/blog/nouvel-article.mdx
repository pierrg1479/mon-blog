## Galeries d'images

### Galerie Justified

La galerie justified organise les images en rangées de hauteur uniforme, créant un effet visuel harmonieux.

<Gallery type="justified" images={mountainImages} />

### Galerie Masonry

La galerie masonry crée une disposition en colonnes type Pinterest, idéale pour des images de hauteurs variées.

<Gallery type="masonry" images={mountainImages} />

### Galerie Grid

La galerie grid affiche les images dans une grille régulière avec un ratio d'aspect constant.

<Gallery type="grid" images={mountainImages} />

## Listes

### Liste non ordonnée

* Item 1
* Item 2
* Item 3
  * Sous-item 3.1
  * Sous-item 3.2

### Liste ordonnée

1. Premier item
2. Deuxième item
3. Troisième item
   1. Sous-item 3.1
   2. Sous-item 3.2

### Liste de tâches

- [x] Tâche complétée
- [ ] Tâche en cours
- [ ] Tâche à faire

  <summary>Click to expand details</summary>

  <p>You can include any Markdown syntax here:</p>

    <li><strong>Bold text</strong></li>
    <li><code>Code</code></li>
  </ul>
